# GitHub Pages 浏览器版

该目录提供不依赖服务端的静态入口，构建命令为 `npm run build:pages`，产物输出到 `pages-dist/`。

- 保留三轮 10 / 15 / 20 题的自适应流程与每轮报告。
- 首次开始探索时用趣味人物卡收集昵称与四项包容式性别选择，不显示登录或用户管理入口；资料与报告只保存在浏览器 `localStorage`，旧昵称会自动预填。
- 直接访问测评路由也不能跳过人物卡；保存报告会记录当次人物资料快照，旧档案仍可正常打开。
- v3 报告包含“容易被谁点亮 / 谁更容易被你吸引 / 心动 vs 久处”的双向伴侣吸引画像；性别不参与计算。
- 每轮阶段报告和已保存报告都支持保留配色地打印或另存为 PDF。
- 题目页使用直白提示、同区展示情境与题干，并在手机上优先展示问题和选项。
- 每题支持“主选 + 可选次选”，解决两个答案都像自己的情况，同时保留结果区分度。
- 生产版的 ChatGPT 登录、D1 数据库及跨用户管理员后台仍由 Sites 部署承载；GitHub Pages 无法运行这些服务端能力。
